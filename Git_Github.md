# Git&Github

---

## Git

常用命令：

1. git config --global User.name "name"
2. git config --global User.email "email"
3. git remote add origin "branch"
4. git commit -m "Initial commit"
5. git add .
6. git push -u origin branch
7. git checkout branch
8. git merge branch
9. git reflog
10. git reset --hard [index]
11. git init
12. git --version
13. git pull

## Github
